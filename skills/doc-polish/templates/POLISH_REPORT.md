## Score Calculation Reference

```
Starting Score: 100

Layer 1 Deductions:
- Per issue: -2 points
- Maximum: -20 points

Layer 2 Deductions:
- High priority: -5 points each (max -25)
- Medium priority: -2 points each (max -15)
- Low priority: -1 point each (max -10)

Layer 3 Deductions:
- Per warning: -3 points (max -30)

Final Score: max(0, calculated_score)
```

### Score Interpretation

| Score Range | Assessment | Action |
|-------------|------------|--------|
| 90-100 | Excellent | Ready to publish |
| 75-89 | Good | Minor improvements suggested |
| 60-74 | Fair | Review recommended before publishing |
| 40-59 | Needs Work | Significant improvements needed |
| 0-39 | Poor | Major revision required |

---

## Usage Notes

1. **Template Variables**: All `{{VARIABLE}}` placeholders are replaced at runtime
2. **Conditional Sections**: `{{#IF}}...{{/IF}}` blocks only render if condition is true
3. **Loops**: `{{#EACH}}...{{/EACH}}` iterates over arrays
4. **Escaping**: Use `\{{` to render literal braces

This template ensures consistent, comprehensive quality reports across all doc-polish runs.
